Ruby скрипт для парсингу сайту

Сайт - список усіх бібліотек з Вікіпедії.
URL: https://en.wikipedia.org/wiki/List_of_Libraries

Програма використовує гем Nokogiri для зручної роботи з HTML (гем можна звстосовувати також для роботи з XML).
Програма, після виконання, створює результуючий CSV файл в який записуються порядковий номер та назви бібліотек.
